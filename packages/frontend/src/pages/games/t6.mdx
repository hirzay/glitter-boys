# Black Ops 2 Zombies

1. Download the [game files](https://files.sjer.red/games/t6.zip) (14GB).
1. Create an account on the [Plutonium forums](https://forum.plutonium.pw/).
1. Follow [sjerred](https://forum.plutonium.pw/user/sjerred) to add them as a friend.
1. Extract `t6.zip`.
1. Run the `plutonium` executable.
1. Once plutonium is running, look on the left sidebar for an icon labeled "T6 Zombies". Click the icon.
1. Click the "setup" button. In the popup, select the `t6` folder with the game files in it. It should be the same `t6` folder with the `plutonium` executable.
1. Click the "Play" button to start the game.
